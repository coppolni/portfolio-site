Star Social

The idea for the creation of this social media clone came from one of my bootcamps. The features of this site are similar to those you would find on social media platforms. Users can sign up, create groups or join other groups, post thoughts, and comment on other posts.

A few key features of Star Social are below, along with additional tools used.

Please visit my GitHub for access to this repository and more.

Features

  • User-friendly interface
  • Create groups, post, and comment
  • Simple user profile - user handle similar to Twitter
  • Form validation

Dev Tools

  • HTML, CSS, JavaScript, Bootstrap
  • Python - Django Framework
  • Git, GitHub, Atom
